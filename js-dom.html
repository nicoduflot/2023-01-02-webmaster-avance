Document Object Model

le but du DOM est de manipuler le HTML.
Pour manipuler le document, on va avoir besoin de 2 choses :

  • Evénement : Déclencher une action
  • Action : Modifier, ajouter, supprimer des éléments du dom

Pour utiliser les actions, on va devoir attendre que le document soit chargé.
